# Двухэтапный конвейер распознавания предметов Backpack Hero (YOLOv8-seg + ResNet50)

Этот проект реализует высокоэффективную двухэтапную архитектуру распознавания и сегментации инвентарных сцен игры.

---

## 🏗️ Архитектура

1.  **Этап 1: Сегментация общего вида (YOLOv8-seg)**
    *   **Вход:** Обрезанная игровая сетка ($1080 \times 720$ пикселей).
    *   **Модель:** YOLOv8m-seg.
    *   **Классы (2):** `bag` (сумка), `item` (предмет).
    *   **Задача:** Сверхточный поиск координат полигонов масок всех объектов (без определения их конкретных названий).

2.  **Этап 2: Тонкая классификация (ResNet50)**
    *   **Вход:** Изолированное изображение предмета ($224 \times 224$ пикселей), где все фоновые пиксели за пределами маски заменены на черный цвет.
    *   **Модель:** PyTorch ResNet50.
    *   **Классы (865):** Все виды сумок и предметов.
    *   **Задача:** Точное определение названия предмета по его текстуре.

---

## 📂 Новые созданные файлы

*   `datasetgen/export_yolo_2classes.py` — Генератор датасета сегментации для YOLOv8m-seg (вырезает игровую доску 1080x720 и размечает на 2 класса).
*   `datasetgen/export_classifier.py` — Генератор датасета для классификатора (вырезает и аугментирует 865 классов изолированных предметов).
*   `train_kaggle.py` — Обучающий скрипт для запуска в один клик на Kaggle GPU.
*   `pipeline.py` — Сквозной скрипт инференса (распознавание полной игровой сцены).

---

## 🚀 Инструкция по запуску

### Шаг 1. Локальная генерация датасетов
Перед загрузкой на Kaggle необходимо сгенерировать обучающие выборки:

```bash
cd Red

# 1. Генерируем датасет сегментации (например, 3000 сцен)
python datasetgen/export_yolo_2classes.py --n 3000 --val-frac 0.1

# 2. Генерируем датасет классификатора (865 классов, по 50 вариантов на класс)
python datasetgen/export_classifier.py --n-train 40 --n-val 10
```

После этого у вас появятся папки:
*   `dataset_cropped_2classes/` (датасет YOLO)
*   `classifier_dataset/` (датасет классификатора)

Заархивируйте их перед загрузкой на Kaggle.

---

### Шаг 2. Обучение на Kaggle
1.  Создайте новый **Kaggle Notebook**.
2.  Подключите GPU (Tesla T4) в настройках сессии.
3.  Загрузите ваши заархивированные датасеты в Kaggle.
4.  Скопируйте содержимое скрипта `train_kaggle.py` в ячейку ноутбука.
5.  Обновите пути к датасетам (строки `data_yaml_path` и `dataset_dir`), указав пути к загруженным ресурсам в Kaggle (например, `/kaggle/input/...`).
6.  Запустите выполнение. По окончании обучения скачайте два файла весов:
    *   `yolov8m_2classes.pt` (из папки результатов тренировки YOLO)
    *   `resnet50_classifier_best.pth` (веса классификатора)

---

### Шаг 3. Инференс и распознавание (pipeline.py)
Поместите скачанные файлы весов и файл списка классов `classes_classifier.txt` в папку проекта и запустите распознавание любого скриншота:

```bash
python pipeline.py --img path_to_screenshot.png --yolo yolov8m_2classes.pt --classifier resnet50_classifier_best.pth
```

Скрипт автоматически вырежет игровую зону, сегментирует сумки и предметы, изолирует их, распознает точные названия и сохранит красивый результат в файл `result_output.png`.
